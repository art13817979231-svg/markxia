# 🌟 英小星 - 小学中学英语学习平台

一款面向小初高学生的英语单词学习网站，支持多种练习模式。

## ✨ 功能特点

- 📖 **单词练习** — 听音频，拼写单词
- 📝 **句子练习** — 听音频，拼写句子
- 🃏 **翻卡模式** — 看中文，想英文
- ✅ **选择题** — 四选一，快速练习
- 📊 **学习统计** — 追踪学习进度
- 🎨 **液态玻璃设计** — 现代化 UI

## 📚 词库覆盖

| 学段 | 词本数 | 总词汇量 |
|------|--------|----------|
| 小学（3-6年级） | 8 册 | 1200+ |
| 初中（7-9年级） | 3 册 | 1800+ |
| 高中（10-12年级） | 3 册 | 900+ |
| 句子练习 | 3 级 | 90 句 |

## 🚀 在线访问

部署于 [Vercel](https://vercel.com)

## 📦 技术栈

- 纯 HTML/CSS/JavaScript
- PWA 支持（离线可用）
- Web Speech API（语音合成）
- LocalStorage（进度存储）
